Java - Sigmoidal Java

Detección de Spam

Perceptrón con función sigmoidal para clasificar correos electrónicos como spam o no spam.

  • Función de activación: Sigmoide
  • Problema: Clasificación de texto
  • Dataset: Características de correos
  • Salida probabilística entre 0 y 1
// Función sigmoidal public double sigmoid(double x) { return 1 / (1 + Math.exp(-x)); } // Derivada de la sigmoidal public double sigmoidDerivative(double x) { return sigmoid(x) * (1 - sigmoid(x)); }

# Función de activación Softmax def softmax(self, z): # Para evitar overflow, restamos el máximo z_exp = np.exp(z - np.max(z)) return z_exp / np.sum(z_exp) # Predecir probabilidades def predecir_probabilidades(self, x): z = np.dot(x, self.pesos) + self.sesgo return self.softmax(z)
